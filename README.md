# Collaboration Demo Project

This project is used to practice Git and GitHub collaboration workflows.  
It is intended as a simple environment where contributors can experiment with common team development practices such as branching, code reviews, and pull requests without worrying about breaking a real production project.

The goal of this repository is to help developers become more comfortable with collaborative development using Git and GitHub.

## How to contribute

- Open an issue to describe the change or feature you want to add.
- Create a branch for your work.
- Make your changes and commit them with clear messages.
- Submit a pull request so others can review your work.
- Address any feedback from reviewers and update your pull request if needed.
